# Obvious shortcuts
```
                                       +---------------------+-----------------+
                                       | Windows / Linux     | macOS           |
+--------------------------------------+---------------------+-----------------+
| select whole text                    | Ctrl + A            | Cmd + A         |
| paste selected text                  | Ctrl + V            | Cmd + V         |
| copy selected text                   | Ctrl + C            | Cmd + C         |
| cut selected text                    | Ctrl + X            | Cmd + X         |
| undo edit                            | Ctrl + Z            | Cmd + Z         |
| redo edit                            | Ctrl + Y            | Shift + Cmd + Z |
| delete next character                | Backspace           | Backspace       |
| delete previous character            | Del                 | Del             |
| go one character left                | Left                | Left            |
| go one character right               | Right               | Right           |
| go one line up                       | Up                  | Up              |
| go one line down                     | Down                | Down            |
| add a line break                     | Enter               | Enter           |
+--------------------------------------+---------------------+-----------------+
```
